Add Spicepod - adds a Spicepod to the project

### Usage

```shell
spice add [flags]
```

#### Flags

- `-h`, `--help`   Print this help message

### Examples

```shell
spice add spiceai/quickstart
```

### Additional Example

```shell
spice add spiceai/samplepod
```

Generate the autocompletion script for spice for the specified shell.
See each sub-command's help for details on how to use the generated script.

### Usage

```shell
spice completion [command]
```

Available `command`s

- bash
- fish
- powershell
- zsh

#### Flags

- `-h`, `--help`   Print this help message

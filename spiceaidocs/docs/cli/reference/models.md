Lists models loaded by the Spice runtime

### Usage

```shell 
spice models [flags]
```

#### Flags

  - `-h`, `--help`   help for models

### Examples

```shell
>>> spice models

NAME  FROM                                    DATASETS STATUS
modlz file:/Users/jeadie/Downloads/model.onnx []       Ready
```
